FRT - File Rename Tool
------------------------------------

Renames (newspaper) image files according to selected boundaries and dates.

This tool was written for the European Newspaper project (http://www.europeana-newspapers.eu/).

The main tool is located in the subproject "FileRenameTool".
The project "SebisForms" is a collection of utility forms.
The tool uses GraphicMagick (http://www.graphicsmagick.org/).
